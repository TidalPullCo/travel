# Resort Intelligence Command Center

A global hospitality lookup engine built for strategists.

## 🚀 Deploy on Vercel

1. Create a GitHub repo and upload all files
2. Go to [vercel.com](https://vercel.com)
3. Click “New Project” → Select your repo
4. Choose “Framework: Other”
5. Deploy instantly

## 🧠 Features

- Tiered resort intelligence
- Emotional resonance scoring
- Aviation + real estate overlays
- Exportable conquest reports
- SaaS role logic (Free, Premium, Admin)

## 🛠 Tech Stack

- HTML + TailwindCSS
- Vanilla JS
- JSON seed data
- Vercel hosting
